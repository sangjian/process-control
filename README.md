## 简介

一个轻量级的流程控制框架，用于解决业务代码臃肿以及难于理解的情况，通过流程节点将业务逻辑更加精细化，使业务流程可编排并且能够清晰易懂，同时便于代码复用，能够以更加灵活的方式实现复杂的业务逻辑。
